🆘 COMMON ISSUES & SOLUTIONS
================================================================================

Issue: "could not find driver"
Fix: Make sure MySQL is running and .env DB config is correct

Issue: "Class not found"
Fix: Run composer dump-autoload or composer install

Issue: "View not found"
Fix: Check path: resources/views/... and file names

Issue: "Redis connection refused"
Fix: Start Redis server or change CACHE_STORE to database

Issue: "Migration failed"
Fix: Check DB connection or drop database and re-run migrations

Untuk detail lengkap: Lihat SETUP_GUIDE.md atau RAILWAY_DEPLOYMENT.md
